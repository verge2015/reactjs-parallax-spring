# reactjs-parallax-spring

This is a single page static website using JavaScript framework ReactJS and packages like React-Spring for parallax effect and Material UI mainly for typography.

## Packages
```bash
React-Spring
Material UI
```

## Install Packages
```bash
 npm install
 or
 npm i
```

## Usage
```bash
 npm start
```
