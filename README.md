# reactjs-parallax-spring

This is a single page static website using JavaScript framework ReactJS and packages React-Spring and Material UI.

## Packages
```bash
React-Spring
Material UI
```

## Install Packages
```bash
 npm install
 or
 npm i
```

## Usage
```bash
 npm start
```
